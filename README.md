# dotfiles

This repository exists to manage my dotfiles and bootstrap my workstation.

## Init

On a brand new machine launch this script :

```sh
curl -fsSL https://raw.githubusercontent.com/mehdicopter/dotfiles/main/install.sh | bash
```
